To-Do List Web App

Welcome to my To-Do List web application. This project is a simple and interactive task management app built using HTML, CSS, and JavaScript. It allows users to manage their daily tasks efficiently with an easy-to-use interface.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Features

Add new tasks using an input field

Mark tasks as completed

Delete tasks when no longer needed

Save tasks using browser localStorage

Responsive design for desktop and mobile devices

Clean and minimal user interface
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Technologies Used

HTML5 for structure

CSS3 for styling and layout

JavaScript for dynamic behavior and task management
